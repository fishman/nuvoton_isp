Installation¶

Stable release¶

To install Nuvoton ISP, run this command in your terminal:

$ pip install Nuvoton ISP

This is the preferred method to install Nuvoton ISP, as it will always install the most recent stable release.

If you don’t have pip installed, this Python installation guide can guide you through the process.

From sources¶

The sources for Nuvoton ISP can be downloaded from the Github repo.

You can either clone the public repository:

$ git clone git://github.com/fishman/Nuvoton ISP

Or download the tarball:

$ curl  -OL https://github.com/fishman/Nuvoton ISP/tarball/master

Once you have a copy of the source, you can install into poetry virtual environment with:

$ poetry install
